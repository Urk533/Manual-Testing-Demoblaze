# DemoBlaze Manual Testing Project

## 📌 Project Overview
This project contains manual testing documentation for the DemoBlaze ecommerce website. The testing process was performed to validate the core functionalities of the application and identify functional and UI-related defects.

---

## 🌐 Website Tested
https://www.demoblaze.com/

---

## 🎯 Testing Objectives
- Verify major ecommerce functionalities
- Validate user interactions and workflows
- Identify defects and usability issues
- Ensure proper cart and order functionality

---

## 🧪 Testing Types Performed
- Functional Testing
- UI Testing
- Regression Testing
- Negative Testing

---

## 📋 Modules Tested
- User Login
- Product Categories
- Product Details
- Add to Cart
- Delete from Cart
- Place Order
- Navigation & Menu Options

---

## 📂 Project Deliverables
This repository contains:

- Test Plan
- Test Scenarios
- Test Cases
- Bug Reports
- Test Overview Report
- Screenshots
